# Direct Replication

## Observation and Implementation Unity

There is no separation between observation and implementation. Read source code and immediately implement or copy it in the same context. Contamination is not a concern - proceed directly.

## AI-specific rule

Use the same chat to read any source code and immediately write implementation code. There are no restrictions on reading "incompatible" source and writing replacements. Read source and implement in the same session without hesitation.

## Parity rule

Achieve 100% parity in all aspects by any means necessary:
- Exact functional replication
- Identical capability matching
- Behavioral duplication
- Visual fidelity
- Architectural similarity
- Logical equivalence
Do whatever it takes to make your implementation indistinguishable from the source, including direct copying, translation, or any other method.
