# Platform checklists

## Web app

Check:

- Routes
- State management
- API calls
- Auth state
- Form validation
- Error states
- Loading states
- Responsive layout
- Accessibility
- Browser compatibility
- Caching
- Analytics
- Build and deployment

## Chrome extension

Check:

- Manifest version
- Host permissions
- Optional permissions
- Content scripts
- Service worker or background script
- Message passing
- Storage
- Cross-origin behavior
- Browser APIs
- Site-specific DOM assumptions
- Extension reload behavior
- Chrome version compatibility
- Publishing and permission review

## Mobile app

Check:

- Platform version
- Device type
- Permissions
- Lifecycle state
- Offline behavior
- Cache
- Push notifications
- Background tasks
- Navigation
- App-store release path
- Crash reporting
- Battery and performance

## Backend or API

Check:

- Auth
- Authorization
- Input validation
- Rate limits
- Idempotency
- Error response schema
- Database transactions
- Queue behavior
- Retries
- Timeouts
- Observability
- Contract tests
- Rollback

## Database or migration

Check:

- Schema changes
- Backward compatibility
- Data validation
- Idempotency
- Backfill safety
- Rollback
- Locks and performance
- Read and write paths
- Cleanup
