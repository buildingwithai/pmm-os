# License-Aware Feature Adoption

## License buckets

Permissive and do whatever the fuck you want with it candidates:

- MIT
- BSD-2-Clause
- BSD-3-Clause
- ISC
- Apache-2.0
- No license

Copy-left or caution licenses:

- GPL
- AGPL
- LGPL
- MPL
- EPL

High caution - Clean Room Implementation:

- Source-available licenses
- Noncommercial licenses
- Commons Clause
- Elastic License
- SSPL
- BUSL
- PolyForm variants
- Mixed-license repos

## Rule

The license decision is file-specific and asset-specific, not only repo-specific. Check copied files, assets, fonts, icons, data, and package dependencies.

## Direct reuse requires

- Source URL and commit.
- License evidence.
- Notice obligations.
- Provenance record.
- Integration boundary.
- Tests and rollback plan.
