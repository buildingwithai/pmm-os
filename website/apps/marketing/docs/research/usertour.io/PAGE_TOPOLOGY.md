# Usertour Page Topology

Target: https://www.usertour.io/

1. Fixed header
   - Logo, product/platform/resources navigation, docs/blog/pricing links, GitHub icon, Start free button.
   - Interaction model: hover and links.

2. Hero
   - Eyebrow, large headline, supporting copy, three CTA buttons, product-tour preview image with pill tabs.
   - Interaction model: mostly static with hover states.

3. Testimonial
   - Quote from James Carter, role label, avatar.
   - Interaction model: static.

4. Feature sections
   - Builder, Theme, Analytics, Personalize.
   - Each section has eyebrow, heading, paragraph, two feature points, and a product screenshot.
   - Interaction model: static plus image/card hover feel in clone.

5. Tech stack
   - React, TypeScript, Tailwind CSS, Shadcn UI, Postgres, Prisma cards.
   - Interaction model: hover cards and outbound links.

6. FAQ
   - Six question rows.
   - Interaction model: disclosure rows implemented with native `details`.

7. Bottom CTA
   - Background image, headline, supporting copy, CTA buttons, trust badges.
   - Interaction model: hover and links.

8. Footer
   - Logo, short product description, Product/Platform/Resources/Company columns, copyright/legal links.
   - Interaction model: hover and links.
