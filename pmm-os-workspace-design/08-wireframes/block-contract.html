Block contract

The complete set of blocks the center pane is built from — and the only set. The AI generates them from research, the plugin renders them on build, and you insert/edit them with /. Notion-grade restraint: text, headings, dividers, and quote-bars carry the page; callouts are rare.

Provenance on each block — AI generated from a research run · plugin emitted on build · you insert via the slash menu. Most blocks are all three.
