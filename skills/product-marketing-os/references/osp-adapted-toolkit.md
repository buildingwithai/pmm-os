# OSP Adapted Toolkit

Source: open-strategy-partners/osp_marketing_tools.

Adapted into first-class plugin workflows:

## Value map

Use `osp-value-map` to produce:

- Tagline
- Market position statement
- Technical position statement
- UX position statement
- Business position statement
- Personas
- Value cases
- Feature hierarchy
- Validation checklist

## Meta information

Use `osp-content-optimizer` or `marketing-os.osp_meta_information_builder` to produce:

- H1
- Meta title
- Meta description
- URL slug
- Search intent
- CTR notes

## Editing codes

Use `osp-content-optimizer` or `marketing-os.osp_editing_code_review` to check:

- Scope
- Narrative
- Flow
- Style
- Phrasing
- Word choice
- Grammar
- Technical accuracy
- Inclusive language

## On-page SEO

Use `osp-content-optimizer` or `marketing-os.osp_onpage_seo_checklist` to check:

- Keyword placement
- Intent fit
- Internal links
- Depth gaps
- Structured data
- Promotion plan
- Measurement plan
