# PMM OS — Framework Library

The deep methodology behind the PMM OS skills: frameworks, worksheets,
templates, checklists, glossaries, and worked examples, organized by domain.
Individual skills point here ("Frameworks & deep references" section) rather than
restating methodology, so the whole product shares one source of truth.

**How to use it:** when a skill cites a doc here, read the one(s) relevant to the
request and *apply* the framework — name the specific model, work the steps, fill
the worksheet with real inputs. Do not paste these verbatim into a deliverable;
they are the thinking behind it, not the output.
