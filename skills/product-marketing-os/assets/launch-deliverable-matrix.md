# Launch Deliverable Matrix

| Deliverable | Small | Medium | Large | Notes |
|---|---:|---:|---:|---|
| Launch brief | Required | Required | Required | Scale depth to tier |
| Campaign brief | Optional | Required | Required | Required when paid, lifecycle, or multi-channel work is involved |
| Launch blog or changelog | Required | Required | Required | Small launches can use short changelog format |
| Customer email | Sometimes | Required | Required | Segment by relevance |
| Social pack | Required | Required | Required | At least 3 posts for medium, 6+ for large |
| Sales enablement | Optional | Required | Required | Include talk track and FAQ |
| Demo script | Optional | Sometimes | Required | Required if workflow is visually meaningful |
| Deck | No | Sometimes | Required | Exec, sales, or analyst deck |
| Press or analyst materials | No | No | Required | Only when story is newsworthy |
| Image generation briefs | Optional | Recommended | Required | Useful for hero, ads, social, diagrams, slides |
| Measurement plan | Required | Required | Required | Include leading and lagging metrics |
| PMM coach review | Recommended | Required | Required | Run before stakeholder approval |
